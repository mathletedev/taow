# taow | The Art of War

> An Art of War quote generator

## Installation

NPM:

```bash
npm install taow
```

Yarn:

```bash
yarn add taow
```

## Usage

```ts
import taow from "taow";

const quote = taow();

// "All warfare is based on deception."
console.log(quote);
```
